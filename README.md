# 🌤 Skycast Weather App

A simple browser-based weather app that displays current conditions, hourly and weekly temperature charts, and an interactive map for any city.

## Features

- **City search** – Look up weather by city name
- **Current & tomorrow's forecast** – At-a-glance weather info
- **Charts** – Today's hourly and weekly temperature trends (Chart.js)
- **Interactive map** – Pin the searched city's location (Leaflet.js)

## Project Structure

```
├── index.html   # App markup and library imports
├── style.css    # Styling and background
└── script.js    # Weather logic and API calls
```

## Dependencies

- [Chart.js](https://www.chartjs.org/) – Temperature charts
- [Leaflet.js](https://leafletjs.com/) – Interactive map

## Getting Started

1. Clone or download the repo.
2. Add your weather API key to `script.js`.
3. Open `index.html` in a browser — no build step needed.
